# Body Note Web App Guide

## 목적

`apps/web`의 초기 프론트엔드 엔트리 구조와 실행 방법을 빠르게 확인하기 위한 안내 문서입니다.

## 포함 범위

- Vite + React + TypeScript 기반 웹 앱
- React Three Fiber 기반 7-layer 뷰어 프리뷰
- Zustand 기반 단일 타임라인 스토어
- Tailwind CSS v4 기반 Functional Minimalism 토큰

## 실행

```bash
npm run dev
```
